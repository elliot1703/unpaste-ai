# Unpaste.ai Social Media Prompts for Nano Banana

## Brand Style Guide Summary

### Design Style: Swiss Brutalist
- **Sharp edges everywhere** - zero border-radius on all elements
- **Grid backgrounds** - visible 60px grid lines
- **Offset shadows** - 4-6px solid black drop shadows on interactive elements
- **High contrast** - near-black on off-white, with bold red accents
- **Uppercase typography** - especially for headlines and labels
- **Numbered sections** - use [001], [002] format for labels

### Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Primary Red | #DC2626 | CTAs, accents, highlights |
| Background | #FAFAFA | Off-white base |
| Foreground | #09090B | Text, shadows |
| Muted Gray | #71717A | Secondary text |
| Border Gray | #E4E4E7 | Lines, dividers |

### Typography Rules
- **Headlines:** Bold/Extra-bold sans-serif, tight letter-spacing (-0.04em), ALL CAPS
- **Body text:** Monospace font (Space Mono style), regular weight
- **Labels/Tags:** Monospace, 10-11px, uppercase, wide letter-spacing (0.05-0.1em)
- **Section tags:** Format as [001], [002], etc. in red monospace

### Visual Elements
- Red solid squares (not circles) as bullet points
- Grid pattern backgrounds
- Sharp rectangular buttons with offset shadows
- Simple geometric shapes only

### Brand Voice
- Direct and punchy
- Problem-focused, anti-admin
- Technical but accessible
- Action-oriented
- No fluff or corporate speak
